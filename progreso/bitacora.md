# Bitácora

Registro cronológico de sesiones. Una entrada por sesión, breve.

---

# Estructura de entrada

```
## AAAA-MM-DD

**Módulo**: X.Y - nombre
**Duración aproximada**: ...
**Resultado**: consolidado | parcial | abierto
**Sistema-anchor**: [breve]
**Comentario**: [una a tres líneas sobre lo que pasó]
**Próximo paso**: [...]
```
