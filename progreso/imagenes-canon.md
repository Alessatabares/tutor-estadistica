# Imágenes-de-sistema canonizadas

Catálogo de instancias específicas de sistemas-anchor que produjeron insight para Alessa. Cuando una imagen funciona repetidamente como base para un concepto, se canoniza acá.

Diferencia con `metodologia/glosario-metaforas.md`: ese es el catálogo de PATRONES (estructuras reusables). Este es el catálogo de INSTANCIAS (imágenes concretas con su concepto asociado).
