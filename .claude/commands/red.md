Producí una vista navegable de la red conceptual, con énfasis en lo que falta.

# Pasos

1. Leé `progreso/red-conceptual.md`.
2. Mostralo agrupado por nivel del temario (1-5).
3. Para cada nodo, mostrá: estado (abierto/consolidado/débil/huérfano), conexiones salientes, conexiones entrantes.
4. Después del listado, agregá:

```
## Conceptos huérfanos
[nodos sin conexiones articuladas]

## Ramas poco desarrolladas
[áreas de la red con pocos nodos o pocas conexiones]

## Conexiones esperadas y aún no establecidas
[ej. si vimos varianza pero todavía no la conectamos con dispersión, marcarlo]
```

# Por qué

El objetivo del aprendizaje no es la lista de conceptos vistos sino la red de conexiones articuladas. La red revela el progreso real, no la cantidad de módulos cubiertos.
